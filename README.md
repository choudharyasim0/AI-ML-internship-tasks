## Task 6: House Price Prediction
### Objective
To predict house prices using property features such as bedrooms, bathrooms, and area size.

### Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

### Machine Learning Model
- Multiple Linear Regression

### Work Performed
- Loaded real estate housing dataset (`zameen-updated.csv`)
- Cleaned, prepared data, and handled missing values
- Selected important features (`bedrooms`, `baths`, `Area Size`)
- Split dataset into training and testing sets (80/20)
- Trained regression model
- Predicted house prices and evaluated metrics:
  - Mean Absolute Error (MAE)
  - Root Mean Squared Error (RMSE)
  - $R^2$ Score (Variance Explained)
- Created actual vs predicted price visualization scatter plot
- Generated feature correlation heatmap

### Skills Learned
- Regression Modeling
- Feature Selection & Subsetting
- Continuous Variable Data Preprocessing
- Error Metric Evaluation ($MAE$, $RMSE$, $R^2$)
